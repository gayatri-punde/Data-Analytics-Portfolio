# Data Analytics Portfolio

This repository contains my Data Analytics projects created using Excel, Power BI, and real-world datasets.

---

## 1. Vrinda Store Annual Report 2022 – Excel Dashboard

### Project Overview
This project analyzes the annual sales performance of Vrinda Store for the year 2022 using Microsoft Excel.

### Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

### Key Insights
- Women customers contributed the highest sales.
- Most orders came from Maharashtra, Karnataka, and Uttar Pradesh.
- The highest sales occurred in March.

### Files Included
- Vrinda_Store_Annual_Report_2022.xlsx – Excel Dashboard
- Vrinda_Store_Annual_Report_2022.pdf – Dashboard Report
