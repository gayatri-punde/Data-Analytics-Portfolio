# Vrinda Store Annual Report 2022 - Excel Dashboard Project

## 📌 Project Overview
This project analyzes the annual sales performance of Vrinda Store for the year 2022 using Microsoft Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning
- Dashboard Design

## 📊 Key Insights
- Monthly sales trend analysis
- Top performing products
- Sales by category
- Customer behavior insights

## 📁 Files Included
- Excel dashboard file (.xlsx)
- Dashboard PDF preview (.pdf)

## 🎯 Objective
To analyze store sales data and generate meaningful business insights through interactive dashboard visualization.
